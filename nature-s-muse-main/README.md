# ArvyaX AI-Assisted Journal System

A full-stack application for tracking nature-based journaling sessions with AI-powered emotion analysis.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- Google Gemini API Key

### Setup
1. **Backend:**
   - Go to `server/`
   - Create a `.env` file with `GEMINI_API_KEY=your_key_here`
   - Run `npm install`
   - Start with `node index.js`

2. **Frontend:**
   - Go to `client/`
   - Run `npm install`
   - Start with `npm run dev`

## 🛠️ Tech Stack
- **Frontend:** React, Tailwind CSS, Lucide Icons
- **Backend:** Node.js, Express
- **AI:** Google Gemini model 1.5 flash
